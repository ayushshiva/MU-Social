# MU Social - Beta Testing Checklist

## 1. Authentication
- [ ] Sign up with a new email/password.
- [ ] Verify logout and re-login persistence.

## 2. Posts & Content
- [ ] Create a text post.
- [ ] Upload a single image post.
- [ ] Create a carousel post (3+ images) and verify scrolling.
- [ ] Like a post and verify the heart icon/count.
- [ ] Add a comment and verify it appears in the comment thread.

## 3. Stories & Reels
- [ ] Upload a 24-hour story.
- [ ] View your own story to see the viewer list.
- [ ] Scroll through the Reels feed and verify video autoplay.

## 4. Social Graph
- [ ] Search for another tester's username.
- [ ] Follow/Unfollow and verify profile count updates.
- [ ] Block a user (if implemented) or report a post for "Inappropriate Content."

## 5. Real-time Chat
- [ ] Open a chat with a follower.
- [ ] Send a text message and verify the "Sent" checkmark.
- [ ] Observe the "is typing..." indicator from the partner.
- [ ] Verify the "Seen" indicator (DoneAll icon) when the partner opens the chat.
- [ ] Long-press a message and select "Delete for Everyone."

## 6. Live Streaming
- [ ] Start a live broadcast.
- [ ] Have another user join and send a "Gift."
- [ ] Verify the stream ends correctly for both parties.

## 7. Wallet & AI
- [ ] View current balance in the Wallet tab.
- [ ] Test the AI-powered search for specific topics.

## 8. Feedback Template
**Found a bug?** Copy this into your report:
```text
Title: [Feature Area] Brief description
Steps: 1. Open app... 2. Click X...
Result: App crashed / Screen turned white
Expected: Screen should show Y
Severity: Low/Medium/High/Blocker
```
